# Frontend (React)

피싱 방어 서비스의 웹 프론트엔드입니다.

## 초기 셋업

아직 프로젝트가 생성되지 않았습니다. 아래 명령으로 이 폴더 안에서 초기화하세요.

```bash
cd frontend
npm create vite@latest . -- --template react-ts
npm install
```

## 실행

```bash
npm run dev
```

## 브랜치 네이밍

`feat/web-*`, `fix/web-*`, `style/web-*` (자세한 규칙은 루트 [README.md](../README.md) 참고)
