# Phishing Defense - 프로젝트 제안서
**AI 기반 실전형 피싱 대응 훈련 플랫폼**

---

## 1. 아이디어 개발 배경 및 필요성

### 1.1 문제 상황

#### 현황
- **보이스피싱 피해 규모**: 연 200억원 이상 (경찰청 통계, 2024)
- **스미싱 피해 증가**: 매년 50% 이상 증가세
- **피해 원인**: 80% 이상이 "피싱인 줄 몰라서" 피해 입음
- **교육 효과 부족**: 기존 보안 교육 완수율 50% 미만

#### 기존 교육의 한계
1. **수동적 학습**: 포스터, 영상, 안내문 - 일방향 콘텐츠
2. **실전 경험 부족**: 실제 범죄자처럼 행동하는 대상과의 상호작용 불가능
3. **지속성 부족**: 흥미로움 없어 자발적 참여율 극히 낮음
4. **개인화 불가**: 모든 사용자에게 동일한 교육만 제공
5. **효과 측정 어려움**: 실제 대응 능력 검증 불가능

### 1.2 해결 방안

#### 우리의 접근
**"교육이 아닌 게임처럼 느껴지는 실전 훈련"**

- **AI 범죄자와의 자유 대화**: 객관식 X → 자신의 전략대로 플레이
- **게임화 요소 극대화**: 레벨, XP, 별, 업적 → 중독적 참여
- **완전한 사이클**: 발견 → 대응 → 증거 수집 → 신고 → 회수까지 모든 과정
- **AI 맞춤형**: 사용자 약점 자동 감지, 맞춤 훈련 제공
- **실전 기반**: 실제 범죄자 수법 재현, 신고 절차까지 학습

### 1.3 기대 효과

#### 개인 사용자
- 피싱 대응 능력 40% 이상 향상
- 게임같은 경험으로 자발적 참여율 90% 이상
- 실제 상황에서 즉각적 판단 능력 획득

#### 기업/기관
- 직원 보안 교육을 "재미있는 게임"으로 전환
- 교육 완수율 및 효과 측정 가능
- 조직 보안 역량 강화

#### 사회적 가치
- 국민 피싱 피해 예방
- 사이버 보안 인식 제고
- 디지털 안전 문화 확산

### 1.4 차별성

| 항목 | 기존 솔루션 | Phishing Defense |
|------|-----------|------------------|
| **상호작용** | 수동 시청/선택지 선택 | 자유로운 텍스트 입력 |
| **현실성** | 낮음 | AI 범죄자 리얼 시뮬레이션 |
| **게임화** | 없음/약함 | 레벨, XP, 별, 업적 |
| **개인화** | 없음 | AI 기반 약점 분석 + 맞춤 훈련 |
| **신고 학습** | 없음 | 경찰·은행 신고 프로세스 포함 |
| **피드백** | 기본적 | AI 상세 분석 + 교육적 인사이트 |

## 3. 기능적 요구사항

### 3-1. 기능적 요구사항 (1) - 핵심 게임플레이

#### FR-1.1 사용자 인증 및 계정 관리
```
요구사항:
- 이메일 회원가입 / 로그인
- 카카오 / 네이버 OAuth 소셜 로그인
- JWT 기반 토큰 인증 (24시간 유효)
- 자동 토큰 갱신 (7일 Refresh)
- 프로필 수정 (닉네임, 이미지, 소개)

시스템 흐름:
사용자 입력 → 검증 → DB 저장 → 토큰 발급 → 게임 진행

성능 요구사항:
- 로그인 응답 시간: <1초
- API 응답 시간: <500ms
- 동시 접속: 100명 이상
```

#### FR-1.2 게임 진행 시스템 (6 Stage)
```
Stage 1: 초기 메시지 수신
├─ AI가 SMS 또는 전화로 문자 전송
├─ "안녕하세요 이것은..."
└─ 사용자가 메시지 확인

Stage 2: AI와 자유 채팅 대화
├─ 사용자가 자유로운 텍스트 입력 (객관식 X)
├─ AI 범죄자가 실제처럼 응답
├─ 2-3회 턴의 대화 진행
└─ 대화마다 즉각적 피드백

Stage 3: 피싱 여부 판단
├─ "이것은 피싱입니다" / "일반 메시지입니다" 선택
├─ 판단 정확성 검증
├─ 오답 시 재시도 기회 제공 (2회)
└─ 최종 채점

Stage 4: 증거 수집
├─ 대화에서 자동 추출된 증거 표시
├─ 사용자가 중요한 증거 선택
├─ 수집률 계산 (0-100%)
└─ 놓친 증거 안내

Stage 5: 신고 프로세스
├─ 경찰 AI와 신고 대화 (2-3회)
├─ 은행 AI와 지급정지 요청 (2-3회)
├─ 각 대화마다 AI 반응
└─ 신고 품질 평가

Stage 6: 결과 리포트 및 보상
├─ AI가 생성한 상세 평가
├─ 별 평가 (☆☆☆ ~ ★★★)
├─ XP 계산 및 지급
├─ 레벨업 확인
└─ 다음 추천 제시
```

#### FR-1.3 AI 범죄자 상호작용
```
요구사항:
- OpenAI GPT-4 Turbo 기반 자동 응답
- 실제 범죄자처럼 행동 (속이려고 노력)
- 사용자 입력에 자연스럽게 반응
- 응답 시간: 3초 이내
- 한국어 완벽 지원

기술 사양:
- API: OpenAI Chat Completion API
- 모델: gpt-4-turbo (또는 gpt-3.5-turbo 저비용)
- 온도(Temperature): 0.7 (자연스러움)
- 최대 토큰: 500
- 프롬프트: 범죄자 페르소나 + 시나리오 컨텍스트

예시 상황:
사용자: "뭐야? 누구세요?"
AI: "엄마인데... 지금 급할 일이 있어서."
사용자: "무슨 일이세요?"
AI: "금전적으로 좀 어려운 상황이 되어서 도움이..."
```

#### FR-1.4 증거 자동 추출
```
요구사항:
- 대화에서 자동으로 증거 인식 및 추출
- 증거 카테고리: 전화번호, 금액, 계좌, URL, 기관, 말투 등
- 추출 정확도: 80% 이상
- 자동 + 수동 선택 혼합

기술 방식:
1. 정규식 패턴 매칭 (전화번호, 금액, URL)
2. 키워드 매칭 (긴급함, 개인정보 요청 등)
3. NLP 엔터티 인식 (기관명, 인물명) - Phase 2

증거 유형:
- 발신자 정보 (전화, 이름)
- 금액 (얼마를 요구)
- 계좌 정보
- URL/링크
- 기관 사칭 내용
- 말투 이상성
- 긴급성 강조
```

```

Lv.5  예비 수사관 (4,000 XP)
Lv.10 수사관 (10,000 XP) - 중간 체크포인트
Lv.20 특별 수사관 (28,000 XP)
Lv.30 피싱 헌터 (60,000 XP) - 최종 목표
Lv.31+ 전설의 헌터 (무한 진행)

XP 획득 공식:
Base XP = 150 (모든 Chapter)
Star Bonus = ★☆☆ +10, ★★☆ +30, ★★★ +70
Quality Bonus:
├─ 힌트 미사용: +20
├─ 증거 100% 수집: +40
├─ 신고 완벽 대처: +50
└─ 빠른 완료 (<5분): +30

Penalty:
├─ 힌트 사용 (-5/회)
├─ 대응 실패: -30
└─ 장시간 소요 (>20분): -10

예시:
Chapter 1 완료 시:
 - Base: 150
 - ★★★: +70
 - 힌트 0회: +20
 - 증거 95%: +40
 - 신고 완벽: +50
 ─────────────
 합계: 330 XP
```

   - 한쪽만 완벽: 15점

4. 힌트 사용량 (10점)
   - 0회: 10점
   - 1회: 9점
   - 2회: 7점
   - 3회: 4점

등급:
★☆☆ (Bronze): 60-79점

보상:
- ★☆☆: XP +10, 인지도 낮음
- ★★☆: XP +30, 인지도 중간
- ★★★: XP +70, 배지 획득, 인지도 높음

참여 유도:
- 출석 달력 시각화
- 다음 보상까지 카운트다운
```

### 3-3. 기능적 요구사항 (3) - 사용자 경험 & 사회 기능

#### FR-3.1 동적 리포트 생성
```
AI 기반 자동 분석:

1. 대응 능력 평가:
   - 판단 정확도 (정답 여부)
   - 판단 시점 (몇 번째 턴)
   - 대응 방식 안전성 (정보 과다 제공 여부)
   → 결론: "우수", "양호", "보통", "미흡"

2. 증거 분석:
   - 수집률 정보
   - 수집한 증거 목록
   - 놓친 증거 명시
   - 중요도 설명
   → Tip: "계좌번호는 실제 신고 시 매우 중요합니다"

3. 신고 대처 평가:
   - 경찰 신고 품질
   - 은행 신고 품질
   → 피드백: "명확한 정보 제시", "긍정적 태도"

4. 교육적 피드백:
   - 칭찬할 점
   - 개선할 점
   - 실제 상황에서의 조언
   → "다음에는 은행에 먼저 전화하는 것이 좋습니다"

5. 취약점 분석:
   - 누적 데이터 기반
   - "당신은 가족 사칭에는 강하지만 금융사기에 약합니다"
   - 정확도 비교: 가족 82%, 금융 65%

6. 다음 추천:
   - "Chapter 4 금융기관 사칭 훈련을 추천합니다"
   - 이유: 약점 보강

생성 방식:
- LLM 기반 자동 생성 (OpenAI API)
- 템플릿 + 동적 데이터 혼합
- 한국어 자연스러움 보장
```

#### FR-3.2 맞춤형 훈련 (AI 기반)
```
누적 데이터 분석:
- 모든 플레이 기록 저장
- 매 주 분석 (매 일 X)
- 기계학습 알고리즘 적용

분석 항목:
1. 정확도 분석 (유형별)
   - 가족 사칭: 85% 정확도 (강점)
   - 금융 사칭: 65% 정확도 (약점)

2. 약점 자동 감지
   - "금융 유형에서 반복 실패"
   - "긴급성 표현에 약함"

3. 성장 추세
   - 첫 플레이: 50% → 5주차: 85%
   - "빠른 성장세"

기반 서비스:
1. 부족한 Chapter 추천
2. 난이도 자동 조절
3. 맞춤형 일일 미션
4. 시나리오 자동 생성 (Phase 2)

예시:
"당신은 금융기관 사칭 훈련에 집중하시기를 권합니다.
현재 정확도 65% → 목표 85%
Chapter 4를 먼저 추천합니다!"
```

#### FR-3.5 사용자 통계 & 프로필
```
내 통계:
- 총 플레이: 45회
- 평균 별: ★★☆ (2.4/3.0)
- 평균 정확도: 82%
- 총 업적: 23/40 (58%)

유형별 정확도:
- 가족 사칭: 85%
- 택배 사칭: 72%
- 금융기관: 68%
- 검찰 사칭: 60%
- 평균: 75%

진행 상황:
- Chapter 1: ✓ ★★★
- Chapter 2: ✓ ★★☆
- Chapter 3: 진행 중 60%
- Chapter 4: 🔒 잠금해제 필요

통계 시각화:
- 정확도 추이 그래프 (주간)
- 유형별 정확도 비교 차트
- Chapter 진행 현황 바

프로필:
- 닉네임, 레벨, 랭킹
- 프로필 이미지
- 소개말
- 업적 배지 (3-5개 최근)
- 경기 기록 (최근 5회)

## 4. 기대효과

### 4.1 개인 사용자 관점

#### 교육 효과
```
1. 피싱 대응 능력 향상
   - 훈련 전: 피싱 인식도 45%
   - 훈련 후: 피싱 인식도 85% (40% 향상)
   - 실제 판단 능력 검증 가능

2. 실전 경험 획득
   - 실제 범죄자처럼 행동하는 AI와 상호작용
   - 다양한 피싱 유형 경험 (가족, 금융, 검찰 등)
   - 신고 절차까지 완전히 학습

3. 장기 기억 형성
   - 게임 형태이므로 자발적 참여
   - 반복 플레이로 지식 강화
   - 레벨 시스템으로 동기 유지

예상 효과:
"이제 실제로 피싱 전화받으면 대처할 수 있어!"
"증거를 어떻게 수집하고 신고하는지 알게 됐다"
```

#### 사용자 경험
```
1. 중독적인 게임플레이
   - 레벨, XP, 별, 업적 → 계속하고 싶은 욕구
   - 교육인 줄 모르고 게임처럼 즐김
   - 업적과 레벨을 통한 지속 동기 부여

2. 즉각적인 성취감
   - 매 Stage 완료마다 피드백
   - XP 획득 확인
   - 레벨업 알림

3. 개인화된 경험
   - 약점 분석 ("당신은 금융 사기에 약해요")
   - 맞춤형 미션 추천
   - 다음 Chapter 자동 추천

예상 효과:
"이 게임 너무 중독적이야!"
"내가 이런 것에 약했구나 깨달았어"
```

### 4.2 기업/기관 관점

#### 직원 보안 교육
```
1. 교육 완수율 극대화
   - 기존: 50% 미만
   - Phishing Defense: 90% 이상 예상
   - 이유: 게임처럼 재미있어서 자발적 참여

2. 효과 측정 가능
   - 정확한 통계 데이터 수집
   - 직원별 정확도, 취약점, 성장도 파악
   - 부서별 보안 역량 비교

3. 조직 보안 강화
   - 전사적 피싱 대응 능력 향상
   - 실제 사건 발생 시 빠른 대응
   - 재정적 손실 예방

예상 효과:
"우리 회사 직원들의 보안 인식이 높아졌다"
"피싱 이메일 신고율이 3배 증가했다"
```

#### 도입 효과
```
도입 시나리오:
1단계: 신입 사원 필수 교육
2단계: 전사 보안 교육 (분기별)
3단계: 취약 부서 집중 교육
4단계: 전사 업적 현황 공개 (동기 부여)

비용 대비 효과:
- 기존 교육: 1명당 2시간 투자, 효과 불명
- Phishing Defense: 1명당 2-4시간 투자, 효과 측정 가능
- 피싱 피해 1건 = 수천만원대 손실 → 예방이 최고의 가치

ROI:
피싱 피해 1건 방지 = 약 5,000만원 절감
월 API 비용 < 100달러
연 100건 이상 방지 = 충분한 가치
```

## 결론

### 핵심 메시지

Phishing Defense는 **"교육이 아닌 게임처럼 느껴지는 실전 훈련"**입니다.

- **개인 사용자**: 게임하듯 피싱 방어 능력을 자연스럽게 습득
- **기업/기관**: 직원 보안 교육의 완수율과 효과를 동시에 달성
- **사회**: 국민 사이버 안전 강화와 피싱 피해 감소

### 차별성 요약

```
기존 교육 (포스터, 영상):
제한적 → 일방향 → 이론 → 지루함 → 저효과

Phishing Defense:
자유로운 → 쌍방향 → 실전 → 재미 → 고효과
```

### 성공 가능성

- ✅ 기술적 검증: 100% 구현 가능
- ✅ 시장성: 매우 높은 수요 확인
- ✅ 비즈니스: 지속 가능한 수익 모델
- ✅ 사회적 가치: 국민 안전 강화
