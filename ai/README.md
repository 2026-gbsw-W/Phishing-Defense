# AI (Python)

피싱 탐지 모델 및 관련 파이프라인입니다.

## 초기 셋업

아직 프로젝트가 생성되지 않았습니다. 아래 명령으로 이 폴더 안에서 초기화하세요.

```bash
cd ai
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt  # requirements.txt는 직접 작성
```
